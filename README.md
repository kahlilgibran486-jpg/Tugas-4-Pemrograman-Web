# Portfolio CRUD PHP MySQL

## Nama

Kahlil Gibran Bangun

## Deskripsi

Website portfolio pribadi yang dibuat menggunakan HTML, CSS, JavaScript, PHP, dan MySQL.

## Fitur

* Create Data Prestasi
* Read Data Prestasi
* Update Data Prestasi
* Delete Data Prestasi

## Cara Menjalankan

1. Import file `portfolio_db.sql` ke phpMyAdmin.
2. Letakkan folder project di `xampp/htdocs/`.
3. Jalankan Apache dan MySQL pada XAMPP.
4. Buka `http://localhost/portfolio`.

## Teknologi

* HTML
* CSS
* JavaScript
* PHP
* MySQL
* XAMPP
